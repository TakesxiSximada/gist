# 今年書くネタを決定して書き始める

ネタは流行りに乗っかるのがいい。

- タイトルにバズワードを混ぜる
- タイトルは煽り要素を入れる
- 内容は流行りに乗るのが良い
- 流行りは次をみる
  - Hackernews
  - Google検索
  - 海外含むカンファレンスの発表内容
- セキュリティインシデントとか押さえる
- 新しく入った仕様などを押さえる
